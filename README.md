Wordle-Game.
5 letter words, 6 attempts.
leaderboard with the fastest time.

Python 3.x must be installed.
tkinter must be available(usally comes with Python.
Make sure words.txt, All_the_words.txt and placemate.json are in the same directory as wordle_gui.py
To run the Wordle scipt in the terminal, use one of these commands:
  $python wordle_gui.py
The default window size its 500 X 750.
If you want custom window size:
  $python wordle_gui.py 600 800
